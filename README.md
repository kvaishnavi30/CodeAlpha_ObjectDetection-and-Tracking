# Object Detection and Tracking

## Description
This project was developed as part of the CodeAlpha Artificial Intelligence Internship. It performs real-time object detection and tracking using the YOLOv8 model and OpenCV. The application detects multiple objects through a webcam and tracks them with bounding boxes and labels.

## Features
- Real-time object detection
- Object tracking using YOLOv8
- Live webcam support
- Bounding boxes and class labels
- Fast and accurate detection

## Technologies Used
- Python
- OpenCV
- Ultralytics YOLOv8

## How to Run

1. Install the required libraries:
   pip install -r requirements.txt

2. Run the project:
   python object_detection_tracking.py
